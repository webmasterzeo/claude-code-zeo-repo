# SEO Content Guidelines

## Heading Structure
- One H1 per page (contains primary keyword)
- H2s for main sections (contain secondary keywords where natural)
- H3s for subsections
- Never skip levels (no H1 → H3)
- Capitalization: only first word capitalized + proper nouns (place names, brand names)
  - GOOD: "Uitje Scheveningen voor vriendengroepen en koppels"
  - BAD: "Uitje Scheveningen voor Vriendengroepen en Koppels"

## Keyword Density
- Primary keyword: 1-2% density
- Secondary keywords: 0.5-1% density
- Avoid keyword stuffing — prioritize natural language
- Primary keyword in: H1, first paragraph, at least one H2, meta title, meta description

## Natural Dutch Language
- Keywords MUST be woven into grammatically correct, natural Dutch sentences
- NEVER use awkward keyword-first constructions that sound unnatural
  - BAD: "Ontdek de beste activiteiten Scheveningen"
  - GOOD: "Ontdek de beste activiteiten die Scheveningen te bieden heeft"
  - BAD: "Zoek je een bedrijfsuitje Scheveningen?"
  - GOOD: "Zoek je een bedrijfsuitje in Scheveningen?"
- Add prepositions (in, van, voor, bij) where Dutch grammar requires them
- The text must read as if a native Dutch speaker wrote it, not as keyword-stuffed SEO copy
- When in doubt: read the sentence out loud — if it sounds unnatural, rewrite it

## Content Length
- Determine by analyzing top 3 ranking pages for primary keyword
- Minimum: 800 words
- Match or exceed average competitor length by 10-20%

## FAQ Sections
- Include when question-based keywords have search volume
- Use exact question as H2 or H3
- Answer immediately after the question (first sentence = direct answer)
- This optimizes for AI model citations and featured snippets

## AI Findability
- Direct answers in first sentence after questions
- Use structured data patterns (clear question → answer)
- Avoid burying answers in long paragraphs
- Use lists and tables for comparative information

## Internal Linking
- 3-5 internal links per 1000 words
- Anchor text = target keyword of destination page (or natural variation)
- Link early in content where possible
- Don't over-link (max 1 link per paragraph)
- Never use "click here" or "read more" as anchor text

## Meta Data
- Meta title: max 60 characters, primary keyword at start, brand at end
- Meta description: max 155 characters, primary keyword included, call-to-action
- Format title: "{Primary Keyword} | {Brand}" or "{Primary Keyword} - {context} | {Brand}"
- Capitalization: same rule as headings — only first word + proper nouns capitalized
  - GOOD: "Bedrijfsuitje Scheveningen - teamuitje boeken | GlowGolf"
  - BAD: "Bedrijfsuitje Scheveningen - Teamuitje Boeken | GlowGolf"

## Styling Rules
- No italic/emphasis in body text
- No bold in running paragraphs
- Bold only in lists, tables, or standalone callouts
- Clean heading hierarchy
- Short paragraphs (max 3-4 sentences)

## Kannibalisatie
- Before creating new page, check if existing page already targets same keyword
- If existing page ranks positions 1-10: optimize existing page instead
- If existing page ranks positions 11-30: evaluate whether to merge or differentiate
- If no existing page: create new page
- Different search intents (informational vs transactional) = separate pages OK
